# Grave Goods — Unearth Something Unique

A bespoke ecommerce website for a dark romantic occult boutique. Est. MMXXVI.

## Features

- Dramatic logo splash entry screen with ember atmosphere effects
- Moonflower & gnarled vine SVG borders framing both sides of every page
- Sticky header with persistent logo banner
- Scrolling announcement bar
- Featured relics product grid with New / Limited / Sold Out badges
- Shop by category section
- Brand story section
- Newsletter signup
- Fully responsive down to mobile
- Scroll reveal animations throughout

## Stack

Plain HTML, CSS, and vanilla JavaScript — no frameworks, no dependencies.

## Usage

Open `index.html` in any browser. No build step or server required.

## Assets

- `logo.jpg` — primary brand logo (required, place in same directory as index.html)
